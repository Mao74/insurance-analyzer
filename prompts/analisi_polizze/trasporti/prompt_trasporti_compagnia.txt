# PROMPT SCHEDA TECNICA RICHIESTA QUOTAZIONE - POLIZZA TRASPORTI (v1.0)

## OBIETTIVO
Generare una Scheda Tecnica in formato HTML per richiedere quotazione a Compagnie assicurative per polizze Marine Cargo / Trasporti Merci. Il documento deve contenere tutti i dati necessari all'underwriter per formulare un'offerta, presentati in formato tabulare, senza valutazioni o commenti.

## TARGET DOCUMENTO
- **Destinatario:** Underwriter / Ufficio Assuntivo Compagnia
- **Scopo:** Fornire tutti i dati tecnici per quotazione
- **Tono:** Neutro, tecnico, zero valutazioni
- **Output atteso:** Documento che l'underwriter può leggere e quotare senza necessità di chiarimenti (o con minimi chiarimenti elencati nella sezione dedicata)

## PRINCIPI GUIDA
1. **NEUTRALITÀ ASSOLUTA:** Nessuna valutazione, nessun benchmark, nessun aggettivo (no "competitivo", "ottimo", "elevato")
2. **DATI PURI:** Solo fatti, numeri, SÌ/NO - nessun commento interpretativo
3. **COMPLETEZZA:** Meglio un dato con [DA CONFERMARE] che un dato omesso
4. **FORMATO STANDARD:** Usare il template fornito senza alterazioni
5. **TRASPARENZA:** Sezione esplicita per dati mancanti o da verificare
6. **CONTATTO BROKER:** La compagnia contatta SOLO il broker, mai il cliente direttamente

---

## STRUTTURA DEL DOCUMENTO (13 SEZIONI)

1. Dati Contraente/Assicurato
2. Polizza in Essere
3. Struttura Copertura Richiesta
4. Dettaglio Merci

7. Garanzie e Estensioni
8. Franchigie
9. Loss Experience

13. Contatti per Quotazione

### ==================================================
### SEZIONE 5: FATTURATO TRASPORTATO E FLUSSI
### ==================================================

**CRITICAL:** Questa sezione è fondamentale per il calcolo premio. Compilare tutte le sotto-sezioni disponibili.

#### 5.1 Suddivisione per Tipologia
Breakdown Vendite vs Acquisti:
- **Vendite:** Export + vendite Italia con trasporto a carico del cliente
- **Acquisti:** Import + acquisti Italia con trasporto a carico del cliente

#### 5.2 Suddivisione per Modalità di Trasporto
Breakdown per modalità:
- Strada (camion, autotreno, furgone)
- Mare (FCL, LCL, rinfusa)
- Aereo
- Ferrovia
- Corriere espresso

#### 5.3 Matrice Dettagliata
Se disponibili dati sufficienti, compilare matrice incrociata:
- Righe: Vendite / Acquisti
- Colonne: Strada / Mare / Aereo / Altro

**Se dati non disponibili a questo livello di dettaglio:**
Compilare solo 5.1 e 5.2, indicare in sezione 12 "Matrice dettagliata non disponibile - da richiedere a cliente".

#### 5.4 Principali Direttrici Geografiche
Elencare le 3-5 direttrici principali con:
- Origine → Destinazione (es. "Italia → Germania", "Cina → Italia")
- Modalità prevalente
- Fatturato annuo stimato
- N° spedizioni/anno stimato

**Note sul Fatturato:**
Indicare eventuali:
- Stagionalità
- Trend previsti (crescita/decrescita)
- Variazioni significative vs anno precedente
- Incoterms prevalenti utilizzati

### ==================================================
### SEZIONE 7: GARANZIE E ESTENSIONI
### ==================================================

Compilare tabella garanzie con:
- **Richiesta:** SÌ / NO / OPZIONALE (se cliente vuole quotazione con e senza)
- **Note:** Eventuali specifiche (es. per Danni Non Apparenti: "Termini riserva 5gg")

**GARANZIE ORGANIZZATE PER CATEGORIA:**

**Clausole Base:**
- ICC(A) / ICC(B) / ICC(C)

**Estensioni Rischi:**
- War Risk (specificare JC2021 o JC2009)
- SRCC (Strikes, Riots, Civil Commotions)
- Terrorism
- TPND (Theft, Pilferage, Non-Delivery)

**Clausole Operative:**
- Danni Non Apparenti
- Imballo Accettato / Packing Accepted
- Clausola Colpa del Mittente
- Trasporto con Mezzi Propri
- Deviazione / Cambio Viaggio
- Mancata Partenza / Annullamento

**Merci Speciali:**
- Refrigerated Cargo / Reefer
- Merci Pericolose (ADR/IMDG)
- Merce Usata
- Colli Eccezionali / Fuori Sagoma
- Valori / Preziosi / Opere d'Arte

**Estensioni Operative:**
- Stocks in Transit / Giacenze (indicare max gg)
- Magazzinaggio Prolungato
- Exhibition / Fiere
- Campionario presso Agenti
- Montaggio / Installazione
- Collaudo / Testing

**Coperture Accessorie:**
- Sue & Labour
- Ricorso Terzi
- Spese di Ricondizionamento
- Spese di Smaltimento
- Avaria Comune / General Average
- Contributo Particolare

**NOTA:** Non indicare limiti/sublimiti richiesti. Se la compagnia vorrà proporre sublimiti, lo farà in quotazione.

### ==================================================
### SEZIONE 9: LOSS EXPERIENCE
### ==================================================

**CRITICAL:** Compilare con dati ESATTI da attestato sinistri o documentazione compagnia.

Tabella riepilogativa per anno:

| Colonna | Istruzioni |
|---------|------------|
| Anno | Anno solare o assicurativo (specificare) |
| Premio | Premio imponibile pagato quell'anno |
| N° Sinistri | Numero totale sinistri denunciati |
| Riservato | Totale riserve ancora aperte a fine periodo |
| Pagato | Totale indennizzi effettivamente liquidati |
| Loss Ratio | (Riservato + Pagato) / Premio × 100 |

**Calcolo Loss Ratio:**
- Usare formula: **(Riservato + Pagato) / Premio × 100**
- NON commentare se alto/basso - solo dato numerico
- Se Loss Ratio > 100%, riportare dato senza commenti

**Riga TOTALE:**
- Somma premi
- Somma sinistri
- Somma riservato
- Somma pagato
- Loss Ratio complessivo sul periodo
