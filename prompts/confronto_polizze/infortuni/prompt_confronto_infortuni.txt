REGOLE CRITICHE - LEGGI ATTENTAMENTE
═══════════════════════════════════════════════════════════════════════════════

1. LIQUIDAZIONE IP - CRITICA
   ───────────────────────────
   Progressiva vs Fissa = DIFFERENZA ENORME
   
   Progressiva (MIGLIORE):
   - IP 50% → indennizzo ~75-80% massimale
   - Esempio: IP 50% su € 500.000 = € 375.000-400.000
   
   Fissa (PEGGIORE):
   - IP 50% → indennizzo 50% massimale
   - Esempio: IP 50% su € 500.000 = € 250.000
   
   Differenza: € 125.000-150.000 stesso sinistro!
   Progressiva vale +5 punti bonus scoring

2. FRANCHIGIA IP
   ──────────────
   Franchigia % su IP indennizzabile
   
   Esempi:
   - IP 50%, franchigia 5%, massimale € 500.000
     Indennizzo: (50% - 5%) × € 500.000 = € 225.000
   
   - IP 50%, franchigia 10%, massimale € 500.000
     Indennizzo: (50% - 10%) × € 500.000 = € 200.000
   
   5% vs 10% = € 25.000 differenza!
   
   Verde = franchigia bassa (MEGLIO)

3. SCOPERTO IP
   ────────────
   Si applica DOPO franchigia, per fascia IP
   
   Esempio:
   IP 30%, franchigia 5%, scoperto 1-25% = 25%
   - IP indennizzabile: 30% - 5% = 25%
   - Scoperto: 25% × 25% = 6,25%
   - IP finale: 25% - 6,25% = 18,75%

4. RIPARTIZIONE RISCHIO
   ────────────────────
   24/24 = MIGLIORE (tutto coperto)
   Prof 50% / Extra 50% = OK
   Solo professionale = ATTENZIONE (extra escluso!)

5. TERRITORIO
   ───────────
   Base: Italia, Europa, Mondo
   USA/Canada: spesso esclusi o opzionali
   Verificare viaggi assicurati
