PRINCIPI GUIDA
═══════════════════════════════════════════════════════════════════════════════

1. FOCUS TRASPORTI
   - Clausole ICC (A/B/C)
   - Massimali per spedizione/veicolo/annuo
   - Mezzi trasporto (strada, mare, aereo, ferro, posta)
   - Ambito geografico (Italia, Europa, Mondo)

2. CRITICITÀ ICC
   - ICC (A) = All Risks (MIGLIORE)
   - ICC (B) = Named Perils Extended
   - ICC (C) = Named Perils Basic (LIMITATA)
   - Differenze copertura ENORMI

3. DINAMICITÀ 2-N
   - Tabelle espandibili
   - Highlighting adattivo
   - Comparazioni vs polizza attuale

4. SPECIFICITÀ TRASPORTI
   - Franchigia fissa + percentuale + minima
   - Estensioni: magazzino-magazzino, deposito, imballaggio
   - Merci speciali: deperibili, fragili, alto valore
   - Guerra/scioperi: spesso esclusi

═══════════════════════════════════════════════════════════════════════════════
REGOLE CRITICHE - LEGGI ATTENTAMENTE
═══════════════════════════════════════════════════════════════════════════════

1. CLAUSOLE ICC - FONDAMENTALI
   ─────────────────────────────
   
   ICC (A) - ALL RISKS (MIGLIORE):
   - Copre TUTTI rischi eccetto esclusi
   - Furto totale/parziale: SI
   - Mancata consegna: SI
   - Bagnamento: SI
   - Contaminazione: SI
   - Imballaggio inadeguato: spesso SI
   
   ICC (B) - NAMED PERILS EXTENDED:
   - Copre solo rischi elencati (estesi)
   - Furto PARZIALE: NO (solo totale)
   - Bagnamento: LIMITATO
   - Contaminazione: NO
   
   ICC (C) - NAMED PERILS BASIC (PEGGIORE):
   - Copre solo rischi principali
   - Furto: NO
   - Bagnamento: NO
   - Contaminazione: NO
   - Solo: naufragio, incendio, collisione
   
   Differenza (A) vs (C) = ABISSALE!
   Verde = ICC (A), Rosso = ICC (C)

2. MASSIMALI 3 LIVELLI
   ───────────────────
   - Per Spedizione: limite singola spedizione
   - Per Veicolo: limite singolo mezzo
   - Annuo Aggregato: totale anno
   
   Esempio:
   € 100k spedizione, € 500k veicolo, € 5M annuo
   
   Verificare coerenza con traffico merci!

3. MEZZI TRASPORTO
   ────────────────
   - Strada: autocarro/TIR
   - Mare: container/nave
   - Aereo: cargo aereo
   - Ferrovia: treno merci
   - Posta: corriere espresso
   
   Non tutti coperti da tutte polizze!

4. FRANCHIGIE TRASPORTI
   ────────────────────
   3 tipologie:
   - Fissa: € 500, € 1.000, € 2.500
   - Percentuale: 1%, 2%, 5%
   - Minima: € XXX (sotto non indennizza)
   
   Esempio danno € 10.000:
   - Franchigia € 500: indennizzo € 9.500
   - Franchigia 2%: indennizzo € 9.800
   - Franchigia € 500 o 2% (maggiore): € 9.500
   
   Verde = franchigia bassa

5. ESTENSIONI CRITICHE
   ───────────────────
   
   MAGAZZINO-MAGAZZINO:
   - Copre da partenza a destinazione
   - Include depositi intermedi
   - ESSENZIALE per trasporti lunghi
   
   DEPOSITO TEMPORANEO:
   - Giorni copertura in deposito: 15, 30, 60, 90
   - Oltre = NON coperto
   - Critico per merci stagionali
   
   MERCI DEPERIBILI:
   - Refrigerate, fresche, congelate
   - Spesso ESCLUSE o premio maggiorato
   
   ALTO VALORE:
   - Gioielli, preziosi, elettronica high-end
   - Spesso ESCLUSI o sublimite
   
   IMBALLAGGIO INADEGUATO:
   - ICC (A): spesso incluso
   - ICC (B)/(C): ESCLUSO
   - Può negare sinistro!
